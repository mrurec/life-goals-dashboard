Платформа на основе виджетов для отслеживания любых личных целей через динамический настраиваемый дашборд. Стек: **Kotlin/Spring Boot + GraphQL (Netflix DGS) + React/Relay** — архитектура и паттерны соответствуют стандартам Meta Engineering.

## Что это такое?

Life Goals Dashboard — это **платформа виджетов**, а не приложение с фиксированным назначением. Пользователи сами составляют свой дашборд, добавляя, настраивая и размещая экземпляры виджетов. Платформа будет поставляться с тремя встроенными типами виджетов:

- **Подготовка к интервью** — отслеживание задач на кодирование с интервальными повторениями (алгоритм SM-2), темы по системному дизайну и поведенческие истории. Настройка целевой компании, уровня и фокусных тем.
- **Фитнес-трекер** — запись тренировок, измерений тела, еды и рецептов с автоматическим расчётом калорий и КБЖУ. Все цели задаются пользователем.
- **Планировщик бюджета** — отслеживание целей по накоплениям с прогнозированием, мониторинг курсов валют. Валюты, дедлайны и категории расходов полностью настраиваются для каждого экземпляра виджета.

Новые типы виджетов добавляются в три шага без изменения основного кода дашборда.

## Стек технологий

| Уровень | Технология |
|---------|-----------|
| Бэкенд | Kotlin 2.3.20 + Spring Boot 4.0.5 (Spring Framework 7, Jakarta EE 11, Jackson 3, JSpecify) на JDK 25 LTS |
| API | GraphQL (Netflix DGS 11.1) |
| Фронтенд | React 19.2 + TypeScript 6.0 + Relay (react-relay 20.1) |
| Состояние | Jotai 2.x (atom-based преемник Recoil; Recoil архивирован upstream) |
| Сборка | Gradle 9.4 (бэкенд) + Vite 8 / Rolldown (фронтенд) |
| База данных | PostgreSQL 18.3 (async I/O, `uuidv7()`, virtual generated columns) |
| Кэш | Redis 8.6 |
| Миграции | Flyway 11 |
| Авторизация | OAuth2 + JWT (Spring Security 7) |
| Тестирование | JUnit 5 + Testcontainers + React Testing Library + Playwright |
| CI/CD | GitHub Actions + Docker |
